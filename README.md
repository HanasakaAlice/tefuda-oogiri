# 手札大喜利

Socket.io と Express で動くシンプルなオンライン大喜利ゲームです。プレイヤーは手札から回答カードを出し合い、お題に対して一番面白いカードに投票します。ドラフト（配られた候補から手札を作る）と自作手札（自分で5枚入力）の2モードに対応しています。

## セットアップ

```sh
npm install
```

## 起動方法

```sh
node server.js
```

ブラウザで http://localhost:3000 を開き、複数タブや別端末でアクセスすると多人数で遊べます。

## 遊び方（流れ）

- 名前を入力して「参加する」をクリック（2人以上で開始可能）。
- ホストがモードを選択して「ゲーム開始」を押す。
- ドラフトモード: 2枚の候補から1枚ずつ選び、手札5枚を作成。
- 自作手札モード: 回答を5つ入力して送信。
- ラウンド開始: 手札から1枚出す → 全員出したら投票フェーズ → 結果表示。
- 規定ラウンド終了後、スコア順に最終結果を表示。ホストは「もう一回遊ぶ」でロビーに戻せます。

## 開発メモ

- フロントエンド: `public/index.html`（Socket.io クライアント＋素の DOM 操作）。
- サーバー: `server.js`（ゲーム状態管理、イベント配信）。
- 依存: express, socket.io（npm で自動インストール）。
